# 🎸 Στάσιμο κύμα σε χορδή
(Κοιλία στο ένα άκρο – Δεσμός στο άλλο)
Διαδραστική εκπαιδευτική εφαρμογή Φυσικής για τη μελέτη στάσιμων κυμάτων σε χορδή με:
κοιλία στο x = 0,
δεσμό στο x = L,
μόνο περιττές αρμονικές.
Η εφαρμογή είναι σχεδιασμένη για χρήση σε:
σχολική τάξη,
projector,
tablet ή κινητό,
GitHub Pages

---

## 📐 Φυσικό μοντέλο

Μήκος χορδής: **L = 0.65 m**  
Ταχύτητα διάδοσης κύματος: **u** (εξαρτάται από την τάση)

### ✅ Επιτρεπτές συχνότητες

fₙ = (2N + 1)·u / (4·L)   (N = 0, 1, 2, …)

Παράγονται μόνο οι **περιττές αρμονικές**: 1η, 3η, 5η, 7η, …

### ✅ Εξίσωση στάσιμου κύματος

y(x,t) = 2A·cos(2πx/λ)·sin(ωt)

με:
- λ = 4L / (2N + 1)
- ω = 2πf

Η μορφή αυτή εξασφαλίζει:
- κοιλία στο x = 0
- δεσμό στο x = L

---

## 🎛️ Χειριστήρια

- **Αρμονική N (περιττές)** – επιλογή αρμονικής
- **Ταχύτητα διάδοσης u (m/s)** – μεταβολή συχνότητας
- **Ήχος** – αναπαραγωγή καθαρής συχνότητας f
- **Slow motion** – επιβράδυνση παρατήρησης για υψηλές αρμονικές

⚠️ Το slow motion **δεν αλλάζει τη φυσική**, μόνο την οπτική παρατήρηση.

---

## 🧠 Slow motion – διδακτική σημασία

Σε υψηλές αρμονικές η περίοδος είναι πολύ μικρή, οπότε:
- το μάτι βλέπει μόνο στιγμιότυπα
- η χορδή φαίνεται ακίνητη

Το slow motion λειτουργεί όπως:
- στροβοσκοπική παρατήρηση
- time‑base σε παλμογράφο
